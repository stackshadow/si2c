About

This library provides an software-based i2c-slave library to communicate over the i2c-bis.

Known working situations:

I2C-Master AVR MHz I2C-Speeds
RaspberryPi B Atmega 8 16Mhz 9600Baud

Configuration

Before you use the library, you need to setup it.
See here on how to do that

Register

The si2c-library predefine registers from where the Master can be read or write to.
You setuped the amount of aviable registers in the configuration header
The slave handle read-write access to this registers automatically.

Example

For an si2c-slave example, see here.

Compile

You can use the provided codeblocks-project or just navigate in an terminal
to the directory where lib/si2c is placed and run: make -f si2c.make Release
